Payette is an open source project licensed under the MIT license. A copy of the “stable” version may be obtained by downloading it here, and the development version here.

If you are interested in developing Payette, contact Tim Fuller about obtaining access to the Payette source repository.

For students in the School of Engineering at the University of Utah, a clone of Payette repository is maintained at lenny.eng.utah.edu:/csm/local/git/Payette.git and can be obtained by

git clone cade_lab_user_name@lenny.eng.utah.edu:/csm/local/git/Payette.git Payette
